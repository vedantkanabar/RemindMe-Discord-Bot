This is a Hackathon 2023 project for our group developing a discord bot for a birthday reminder using Node.js and 
Discord.js library. This program features a functionality to set a birthday reminder, birthday message generator, 
and a timer to say happy birthday. We implemented Twilio API in order to send an SMS reminder at the scheduled date 
and OpenAI API to generate a custom birthday message.
